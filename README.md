# Role Name

Push X.509 certificates to servers and using the structure from certbot for deployment.

## Requirements

None.

## Role Variables

Default variables are set in `defaults/main.yml`.

## Dependencies

No dependency on other Ansible Galaxy roles.

## Example Playbook
